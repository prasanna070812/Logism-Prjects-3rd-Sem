
# Digital Logic Circuits ‚Äì Logisim Projects

This repository contains multiple Logisim circuit files created while learning and practicing **Digital Logic Design** fundamentals. Each project represents a core concept in combinational logic, Boolean algebra, and gate-level circuit implementation. The goal of this collection is to understand how digital systems work internally by designing, building, and testing circuits from basic gates to arithmetic units.
